# Rose Chang — Personal Homepage

Personal homepage of Rose Chang (Mi Chang), Senior Researcher at ETRI.
AI-driven UX · HCI · Autonomous Vehicle & Mobility.

- Single-file static site (`index.html`) — illustrations and web font are embedded as data URIs, no external requests.
- Pages (About / Publications) are switched via hash routing (`#/about`, `#/publications`).
- Illustrations generated with Nano Banana (Gemini); display font is Playfair Display (Latin subset, OFL).
- Hosted on GitHub Pages.
